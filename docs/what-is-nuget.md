<span data-ttu-id="8ebb5-104">Důležitým nástrojem pro moderní vývojovou platformu je mechanismus, pomocí kterého můžou vývojáři vytvářet, sdílet a využívat užitečný kód.</span><span class="sxs-lookup"><span data-stu-id="8ebb5-104">An essential tool for any modern development platform is a mechanism through which developers can create, share, and consume useful code.</span></span> <span data-ttu-id="8ebb5-105">Tento kód je často obsažen do balíčku "Packages", které obsahují zkompilovaný kód (jako knihovny DLL), spolu s dalším obsahem potřebným v projektech, které využívají tyto balíčky.</span><span class="sxs-lookup"><span data-stu-id="8ebb5-105">Often such code is bundled into "packages" that contain compiled code (as DLLs) along with other content needed in the projects that consume these packages.</span></span>

<span data-ttu-id="8ebb5-107">Jednoduše řečeno, balíček NuGet je jeden soubor zip s `.nupkg` příponou, která obsahuje zkompilovaný kód (DLL), další soubory související s tímto kódem a popisný manifest, který obsahuje informace, jako je číslo verze balíčku.</span><span class="sxs-lookup"><span data-stu-id="8ebb5-107">Put simply, a NuGet package is a single ZIP file with the `.nupkg` extension that contains compiled code (DLLs), other files related to that code, and a descriptive manifest that includes information like the package's version number.</span></span> <span data-ttu-id="8ebb5-108">Vývojáři s kódem ke sdílení vytváření balíčků a jejich publikování na veřejném nebo privátním hostiteli.</span><span class="sxs-lookup"><span data-stu-id="8ebb5-108">Developers with code to share create packages and publish them to a public or private host.</span></span> <span data-ttu-id="8ebb5-109">Příjemci balíčku získají tyto balíčky od vhodných hostitelů, přidají je do svých projektů a pak volají funkce balíčku ve svém kódu projektu.</span><span class="sxs-lookup"><span data-stu-id="8ebb5-109">Package consumers obtain those packages from suitable hosts, add them to their projects, and then call a package's functionality in their project code.</span></span> <span data-ttu-id="8ebb5-110">Samotný NuGet potom zpracuje všechny mezilehlé podrobnosti.</span><span class="sxs-lookup"><span data-stu-id="8ebb5-110">NuGet itself then handles all of the intermediate details.</span></span>
